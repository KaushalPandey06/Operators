operators

In the C programming language, operators are symbols that perform operations on variables and values. Each operator serves a specific purpose in manipulating data and controlling program flow.


Arithmatical Operators

In C programming, arithmetic operators perform mathematical operations on variables and values.

Addition (+): Adds two operands.
int sum = a + b;
Subtraction (-): Subtracts the right operand from the left operand.
int difference = a - b;
Multiplication (*): Multiplies two operands.
int product = a * b;
Division (/): Divides the left operand by the right operand.
float quotient = (float)a / b; // Ensure one operand is float for accurate division.
Modulus (%): Returns the remainder of the division of the left operand by the right operand.
int remainder = a % b;

Unary Operators

Unary operators works on a single operand( means variable)There are two type of unary operator in C-language.

Assignment Operators

In C programming, the assignment operators is used to assign a value to the variable.

Simple Assignment (=): Assigns the value on the right to the variable on the left.
int x = 5;
Addition Assignment (+=): Adds the value on the right to the variable on the left.
x += 3; // Equivalent to x = x + 3;
Subtraction Assignment (-=): Subtracts the value on the right from the variable on the left.
x -= 2; // Equivalent to x = x - 2;
Multiplication Assignment (*=): Multiplies the variable on the left by the value on the right.
x *= 4; // Equivalent to x = x * 4;
Division Assignment (/=): Divides the variable on the left by the value on the right.
x /= 2; // Equivalent to x = x / 2;
Modulus Assignment (%=): Computes the modulus of the variable on the left by the value on the right.
x %= 3; // Equivalent to x = x % 3;

Relational Operators

In C, relational operators are used to compare values.
Equal to (==): Checks if two values are equal.
Not equal to (!=): Checks if two values are not equal.
Greater than (>): Checks if the left operand is greater than the right operand.
Less than (<): Checks if the left operand is less than the right operand.
Greater than or equal to (>=): Checks if the left operand is greater than or equal to the right operand.
Less than or equal to (<=): Checks if the left operand is less than or equal to the right operand.

<= less than or equal to A <= B

Ternary or Conditional Operators

In C, the ternary operator is a shorthand way of writing an if-else statement.

Its syntax is:
condition ? expression_if_true : expression_if_false;

ternary Operator
condition ? value_if_true : value_if_false;

Logical Operators

In C, logical operators are used to perform logical operations on boolean values (true or false).

&& (Logical AND): Returns true if both operands are true.
if (x && y) { // Code executes if both x and y are true }
|| (Logical OR): Returns true if at least one operand is true.
if (x || y) { // Code executes if either x or y is true }
! (Logical NOT): Returns true if the operand is false, and false if the operand is true.
if (!x) { // Code executes if x is false }
